# better-golf Project Goals

## Overview
The goal of this project is to create a full-stack application that allows users to track their golf game. The application should be able to track scores, rounds, and other golf-related statistics. The application should be able to be used on both desktop and mobile devices.

## Features
- Track golf scores and rounds
- Track golf players and their results
- Track golf shots and results
- Track golf hole-by-hole performance
- Track golf course information
- Track golf-related statistics
- Mobile-first design
- Simple and minimalistic frontend
- Consistent experience between desktop and mobile devices
- Comprehensive logging for debugging and testing


## User Profile
The app is built for golfers who want to track their game and improve their performance.  The ideal user is someone who plays golf regularly, has a smartphone, and wants to analyze their game to identify areas to improve their score.
