## Statistics requirements

* The statistics page should have a filter to show statistics for the last 10, 20, 50, or all rounds.  This filter should default to last 10 rounds
* The statistics page should display the net and gross scores in a line graph for the last N rounds as the user selects in the filter.  THIS SHOULD BE DISPLAYED AS THE FIRST GRAPH ON THE PAGE, BUT AFTER THE BIRDIE/PAR/BOGEY/DOUBLE BOGEY STATISTIC
* The statistics page should show the average birdies, pars, bogeys, and double bogeys for the last N rounds as the user selects in the filter.  THIS SHOULD BE DISPLAYED AS THE FIRST STATISTIC ON THE PAGE
* The statitics page should show the FIR (fairways in regulation) and GIR (greens in regulation) percentages for the last N rounds as the user selects in the filter.  THIS SHOULD BE DISPLAYED AS THE SECOND STATISTIC ON THE PAGE
* The statistics page should show the number of unique courses played for the last N rounds as the user selects in the filter.  THIS SHOULD BE DISPLAYED AS THE THIRD STATISTIC ON THE PAGE
* The statistics page should show the most played course for the last N rounds as the user selects in the filter.  THIS SHOULD BE DISPLAYED AS THE FOURTH STATISTIC ON THE PAGE
