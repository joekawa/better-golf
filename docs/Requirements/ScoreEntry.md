## Score Entry Requirements
* User needs to be able to search for a course by name
* User needs to be able to select a course from the search results
* User needs to be able to select a date for the round
* User needs to be able to select either total score or hole by hole scoring
* If a user selects hole by hole score, then the app should automatically populate the hole numbers and par values from the course data
* If a user selects hole by hole score, then the app should automatically calculate stats and gross score based on the entered hole scores
* If a user selects total score, then the app should only ask for the gross score
* If a user selects total score, the app should also allow the user to enter statistics into the round results (birdies, bogies, etc.)
* The app should calculate net score based on the user's handicap and the course rating and slope
* User needs to be able to enter their score for each hole
* User needs to be able to save their score
* User should have to confirm before saving the score in the app
